# Java FX Music Player by Kalvin Kao
A music player coded in Java.
<br>

Features
- Play/Pause
- Next track
- Rewind Song / Previous track
- Song slider
- Volume slider
- Load multiple .mp3, .wav, .aac audio files
- About
- Exit

Libaries
- JavaFX

Tools used
- Eclipse
- JavaFX SceneBuilder

Images
- Icons from Google material icons
- Icon from freepik at flaticons.com
